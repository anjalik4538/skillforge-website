# 🚀 SkillForge — Learn. Build. Launch.

A modern, responsive **EdTech landing page** built using HTML, CSS, and JavaScript.
SkillForge is designed to help users discover courses, build real-world skills, and get job-ready.

---

## 📌 Features

* 🎯 Clean and modern UI design
* 📱 Fully responsive (mobile + desktop)
* ⚡ Smooth scrolling navigation
* 🎨 Attractive animations & gradients
* 🔍 Course filtering system
* 📩 Email input validation (CTA section)
* 🍔 Mobile hamburger menu

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox + Grid)
* JavaScript (Vanilla JS)

---

## 📂 Project Structure

```
SkillForge/
│── index.html
│── style.css
│── script.js
```

---

## 🚀 How to Run Locally

1. Clone the repository:

```
git clone https://github.com/your-username/skillforge-landing-page.git
```

2. Open the project folder:

```
cd skillforge-landing-page
```

3. Run:

* Simply open `index.html` in your browser
